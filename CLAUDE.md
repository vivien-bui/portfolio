# SharkBeauty Handover Deck — Project Overview

## Purpose

This repo holds the SharkBeauty (Shark Beauty) handover deck for VaynerMedia. It is a deck built for a replacement strategist joining the account, covering everything they need to hit the ground running.

## Deck Design Preferences

- **Style:** SpeedStyle deck format
- **Colour palette:** Frosted Diamond — lean white, clean, not dark/heavy
- **Format:** Google Slides-friendly markdown (each `---` = slide break)
- **Tone:** Professional but approachable; this is onboarding, not a strategy document
- **Icons:** Use text labels when actual icon images aren't available (e.g. `[IG]`, `[TT]`, `[YT]`, `[FB]`)
